# Debian Setup

This repository has files to install programs on Ubuntu-based operating systems

## How to do

* To install all programs, run:

```sh
curl deb.tarsis.cc/all.sh | bash
```

* To install many programs, run:

```sh
curl deb.tarsis.cc/min.sh | bash
```
